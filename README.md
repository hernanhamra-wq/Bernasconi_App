# BernasconiApp
Bernasconi_app es una solución de software de escritorio desarrollada para la gestión digital e integral del inventario de obras y colecciones del Museo Bernasconi (Django/Python). Desarrollado para migrar y centralizar colecciones históricas desde bases de datos Access a MySQL, mejorando la catalogación, seguridad y trazabilidad patrimonial.
